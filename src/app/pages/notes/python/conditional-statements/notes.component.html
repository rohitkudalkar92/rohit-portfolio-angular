• Condition must evaluate to True or False
  • Colon (:) is required after the condition
  • Indentation (4 spaces) defines the code block

Use elif instead of multiple if statements for mutually exclusive conditions - it's more efficient and clearer.

Avoid deep nesting (more than 2-3 levels) - use logical operators or functions instead.
  • and: Both conditions must be True
  • or: At least one condition must be True
  • not: Reverses the boolean value

Use ternary operator for simple assignments. For complex logic, use regular if-else for better readability.
