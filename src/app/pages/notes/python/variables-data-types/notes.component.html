Variable Naming Rules

Essential rules and conventions for naming variables in Python

üìã Basic Rules

  • Must start with letter (a-z, A-Z) or underscore (_)
  • Can contain letters, numbers, and underscores
  • Case-sensitive (age and Age are different)
  • Cannot use Python keywords (if, for, while, etc.)
  • Cannot contain spaces or special characters (@, #, $, %, etc.)
  • Should be descriptive and meaningful

‚úÖ Valid Examples

user_name = "Alice"        # snake_case (recommended)
age2 = 25                  # numbers allowed
_private_var = "hidden"    # leading underscore
PI = 3.14159              # constants
my_list = [1, 2, 3]       # descriptive

‚ùå Invalid Examples

# These would cause SyntaxError:
# 2name = "John"         # Cannot start with number
# user-name = "Alice"    # Cannot contain hyphen
# user name = "Bob"      # Cannot contain space
# if = 25               # Cannot use keywords
# @email = "test"       # Cannot use @ symbol
# user$name = "Alice"   # Cannot use $ symbol
# name% = "Bob"         # Cannot use % symbol
# #count = 10           # Cannot use # symbol

üéØ Naming Conventions

  • Variables & Functions: snake_case
  • Constants: UPPER_CASE
  • Classes: PascalCase
  • Private: _leading_underscore

Variables

Variables are containers for storing data values. Python has no command for declaring a variable.

Python 
# Creating variables
name = "John"
age = 25
height = 5.9
is_student = True

print(name, age, height, is_student)
Output 
John 25 5.9 True

Basic Data Types

Strings (str)

Text data enclosed in quotes

name = "Alice"
message = 'Hello World'
multiline = """This is a
multiline string"""

Numbers

Integers and floating-point numbers

age = 25          # int
price = 19.99     # float
complex_num = 3+4j # complex

Booleans (bool)

True or False values

is_active = True
is_complete = False
result = 5 > 3  # True

None Type

Represents absence of value

value = None
print(value)  # None

Type Checking

Use type() function to check the data type of a variable.

Python 
name = "Alice"
age = 25
height = 5.9
is_student = True

print(type(name))      # <class 'str'>
print(type(age))       # <class 'int'>
print(type(height))    # <class 'float'>
print(type(is_student)) # <class 'bool'>
Output 
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>

Advanced Examples

Type Conversion

Converting between different data types

# String to number
age_str = "25"
age_int = int(age_str)
age_float = float(age_str)

# Number to string
num = 42
num_str = str(num)

# Boolean conversion
print(bool(1))    # True
print(bool(0))    # False
print(bool(""))   # False
print(bool("hi")) # True
Output 
True
False
False
True

Multiple Assignment

Assign multiple variables in one line

# Multiple assignment
x, y, z = 1, 2, 3
name, age, city = "Alice", 25, "NYC"

# Swapping variables
a, b = 10, 20
a, b = b, a  # Now a=20, b=10

# Unpacking
coordinates = (3, 4)
x, y = coordinates
