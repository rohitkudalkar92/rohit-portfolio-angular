Variables and Data Types

Learn about Python variables, strings, numbers, booleans, and basic data types.

Beginner

Variables

Variables are containers for storing data values. Python has no command for declaring a variable.

Python 
# Creating variables
name = "John"
age = 25
height = 5.9
is_student = True

print(name, age, height, is_student)

Basic Data Types

Strings (str)

Text data enclosed in quotes

name = "Alice"
message = 'Hello World'
multiline = """This is a
multiline string"""

Numbers

Integers and floating-point numbers

age = 25          # int
price = 19.99     # float
complex_num = 3+4j # complex

Booleans (bool)

True or False values

is_active = True
is_complete = False
result = 5 > 3  # True

None Type

Represents absence of value

value = None
print(value)  # None

Type Checking

Use type() function to check the data type of a variable.

Python 
name = "Alice"
age = 25
height = 5.9
is_student = True

print(type(name))      # <class 'str'>
print(type(age))       # <class 'int'>
print(type(height))    # <class 'float'>
print(type(is_student)) # <class 'bool'>

Advanced Examples

Type Conversion

Converting between different data types

# String to number
age_str = "25"
age_int = int(age_str)
age_float = float(age_str)

# Number to string
num = 42
num_str = str(num)

# Boolean conversion
print(bool(1))    # True
print(bool(0))    # False
print(bool(""))   # False
print(bool("hi")) # True

Multiple Assignment

Assign multiple variables in one line

# Multiple assignment
x, y, z = 1, 2, 3
name, age, city = "Alice", 25, "NYC"

# Swapping variables
a, b = 10, 20
a, b = b, a  # Now a=20, b=10

# Unpacking
coordinates = (3, 4)
x, y = coordinates

Dynamic Typing

Variables can change types during runtime

# Variable changing types
var = 42        # int
print(type(var))  # <class 'int'>

var = "Hello"   # str
print(type(var))  # <class 'str'>

var = [1, 2, 3] # list
print(type(var))  # <class 'list'>

String Formatting

Advanced string manipulation techniques

# f-strings (Python 3.6+)
name = "Alice"
age = 25
message = f"Hello, name! You are age years old."

# Format method
template = "Name: placeholder, Age: placeholder"
result = template.format(name, age)

# String methods
text = "  Python Programming  "
print(text.strip())     # Remove whitespace 
print(text.upper())     # Uppercase
print(text.replace("Python", "Java"))
